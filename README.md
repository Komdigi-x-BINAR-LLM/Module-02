Selamat datang di *repository hands-on* untuk **Modul 2 ("Fundamental LLM: Dari Pemodelan Bahasa Statistik ke Arsitektur Transformer")** dari program AI Engineer Komdigi!

**Tujuan:**
Repository ini berisi *notebook* Jupyter interaktif (`Modul_2_Tokenisasi_dan_Embedding.ipynb`) yang dirancang untuk membantu Anda **memvisualisasikan** dan **bereksperimen** dengan konsep-konsep teori pra-proses kunci dari Modul 2, yaitu:
